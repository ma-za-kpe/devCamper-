# devCamper-

## By Maku Pauline Mazakpe

This is the recreation of the devcamper project built by Brad Travesy in his [course](https://www.udemy.com/course/nodejs-api-masterclass/learn/lecture/16581786?start=60#questions). The Aim of this project is to sharpen my skills by learning proper nodejs backened development skills, such that i can create backends for my android kotlin apps. The link to the project will be found here. [Project link](https://documenter.getpostman.com/view/3729762/SWEE1urU?version=latest)

## Installing this app

- Clone this github repo in your terminal

- cd devCamper

- npm install

- npm run dev
